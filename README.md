# neurociencia
Neurociencia aplicada a la educaión Proyecto HTML JS CSS Maqueta 

Para Correrlo se puede levantar un servidor en python, se descarga el proyecto, se coloca en una carpeta en especifico y de ahí con CMD se posicionan en esa carpeta y luego ejecutan 

python -m http.server 8000 

Una vez ejecutado ese comando sobre la ruta donde esta el proyecto pueden entrar con 

[http:localhost:8000](http://localhost:8000/)
